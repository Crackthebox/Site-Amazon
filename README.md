# Amazon Biodiversity - Interactive Web Platform

An educational and interactive web application dedicated to the flora of the Amazon Rainforest. This project explores the intersection of biology and technology, featuring an integrated marketplace simulation.

## 🚀 Features
* **Interactive Species Catalog:** Detailed navigation through iconic Amazonian plants (Monstera, Musa, Pitahaya, etc.).
* **Dynamic Authentication System:** Custom-built login/logout functionality to toggle between educational content and the marketplace.
* **Simulated Marketplace:** A dynamic plant shop section where authenticated users can interact with listings.
* **Shopping Cart Logic:** Functional cart with real-time price calculation and state management.
* **Responsive Design:** Styled with CSS using the Montserrat typography for a modern, clean look.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Flexbox/Layouts)
* **Scripting:** JavaScript (ES6+) for DOM manipulation, event handling, and marketplace logic.
* **Assets:** Integration of external Google Fonts and custom iconography.

## 🌍 Sustainability Context
This project was developed to raise awareness about the Amazon's biodiversity, using technology to bridge the gap between complex environmental data and public accessibility.
